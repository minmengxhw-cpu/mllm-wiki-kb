# 上海民盟微信公众号新闻类型写法库

用途：以后用户给到会议、活动、人物采访、两会建言、社会服务、文史纪念等材料时，按本文判断类型并生成接近上海民盟公众号近期风格的文章。

## 一、总体风格

上海民盟公众号近期新闻写作的共同底色是：机关新闻的准确性、微信公众号的可读性、民盟组织身份的稳定表达。

一篇合格稿件通常要同时做到：

1. 事实明确：时间、地点、人物、职务、组织、议程、结果不能模糊。
2. 表达稳妥：政治表达、组织关系、领导职务、活动意义要准确。
3. 可读性强：标题有入口，段落短，信息不绕，必要时用小标题。
4. 价值落点清楚：最终落到履职、组织建设、双岗建功、服务大局、文化传承、民生关切等。

常见语言气质：

- 新闻类：准确、平实、规范。
- 人物类：温暖、具体、有场景。
- 建言类：问题导向、建议具体。
- 文史类：有历史纵深，但判断克制。
- 社会服务类：重成效，温暖但不煽情。
- 预告类：短、直接、问题式入口。

## 二、类型一：事件消息 / 活动新闻

适用材料：活动举行、论坛沙龙、主题教育、培训、座谈、学习会、展览开幕、会客厅活动等。

近期标题样式：

- `在传承中凝聚履职新动能——“我与主委面对面”浦东专场举行`
- `聚焦上海市小班化教学改革试点——“与盟同行•会客厅”（四）举行`
- `民盟市委机关举行“参政为公、实干为民”主题教育专题交流会`
- `以高质量研究赋能精准建言，民盟市委第二批课题开题培训举行`

文章结构：

1. 第一段：时间 + 地点 + 活动名称 + 出席人员 + 活动主题。
2. 第二段：为什么办，背景是什么。
3. 中段：活动流程，谁发言、谁分享、谁交流。
4. 重点段：领导讲话或主旨观点，概括成几个方向。
5. 结尾：活动成效、下一步、参与人员。

导语模板：

> 近日/某月某日，某活动在某地举行。活动围绕……，邀请……参加，旨在……。民盟市委……出席并讲话。

中段模板：

> 活动中，……作主题分享。与会盟员围绕……进行交流，结合自身工作实际谈认识、谈体会、谈建议。

结尾模板：

> 大家表示，将以此次活动为契机，进一步……，在……中展现民盟担当、贡献民盟智慧。

写作要点：

- 不要把所有发言人逐一铺开，除非材料要求。
- 有领导讲话时，优先概括“肯定 + 希望/要求 + 下一步”。
- 没有强新闻结果时，用“交流收获”收束。
- 标题可比正文活泼，但正文仍要稳。

需要用户提供：

- 活动全称、时间、地点、主办/承办单位。
- 出席领导及职务。
- 主要议程。
- 讲话要点。
- 活动亮点或成效。
- 参与对象和人数。

## 三、类型二：会议报道

适用材料：主委会议、常委会议、全委会议、工作会议、专题协商会、机关建设会等。

近期标题样式：

- `民盟上海市委召开十六届四十八次主委会议`
- `中共上海市委召开专题协商座谈会，围绕党风廉政建设和反腐败工作听取党外人士意见建议`
- `以实干进取的精神迎接上海民盟80周年——民盟市委召开机关建设会`

文章结构：

1. 第一段：会议召开事实。
2. 第二段：会议审议/传达/研究事项。
3. 第三段：主要讲话。
4. 第四段：会议要求。
5. 结尾：参会人员。

常用句式：

- `会议传达学习了……`
- `会议审议通过了……`
- `会议指出，……`
- `会议强调，……`
- `会议要求，……`

写作要点：

- 会议稿要少用文学化表达。
- “指出、强调、要求”层级要分清。
- 会议事项多时，用“会议还研究了……”收束。
- 会议结果比氛围更重要。

需要用户提供：

- 会议名称和届次。
- 时间地点。
- 主持人、出席人、列席人。
- 审议/传达/研究事项。
- 主要讲话原意。
- 会议通过或形成的结果。

## 四、类型三：基层换届 / 组织建设新闻

适用材料：区委、基层委员会、总支、支部换届，新组织成立，盟员之家揭牌。

近期标题样式：

- `申康中心、仪电系统、上海电机学院三个民盟基层组织完成换届`
- `民盟黄浦区第四次代表大会召开，方研翔当选主委`
- `上海体育大学、上海开放大学两个高校民盟组织完成换届`
- `上海民盟首个数智产业特色“盟员之家”揭牌成立`

文章结构：

1. 导语：今年是换届年/某组织召开代表大会/完成换届。
2. 大会事实：时间、地点、出席领导。
3. 工作报告：上一届工作回顾。
4. 选举结果：新一届班子、主委、副主委。
5. 讲话要求：政治交接、履职交接、作风交接、组织活力。
6. 结尾：参会人员、兄弟党派贺词、下一步。

常用表达：

- `严格组织程序，严守时间节点，平稳有序开展换届工作。`
- `大会听取并审议了上一届委员会工作报告。`
- `选举产生了新一届委员会。`
- `某某当选主任委员/主委。`
- `希望新一届班子深化政治交接、强化履职担当、加强自身建设。`

写作要点：

- 选举结果必须准确。
- 多个组织合写时，每个组织用同一结构，避免详略失衡。
- 讲话内容可高度概括，但“三点希望/三点要求”要完整。
- 不要写成单纯名单公告，要有组织建设意义。

需要用户提供：

- 组织名称和大会届次。
- 上一届主委、新一届主委和班子名单。
- 出席领导及职务。
- 工作报告亮点。
- 选举程序和结果。
- 讲话要点。

## 五、类型四：调研 / 走访 / 考察新闻

适用材料：领导带队调研、课题组调研、兄弟省市民盟来沪交流、走访盟员单位、专题考察。

近期标题样式：

- `民盟市委专项课题组赴闵行调研上海乡村优质生活空间建设情况`
- `陈群率队赴新虹桥研创中心调研，共绘脑机接口新蓝图`
- `民盟市委赴上海市教育科学研究院和盟员企业开展专题调研`
- `江苏民盟来沪调研座谈`

文章结构：

1. 导语：谁带队，赴哪里，围绕什么主题调研。
2. 调研现场：参观、听介绍、座谈交流。
3. 问题意识：调研关注的重点。
4. 交流观点：双方发言或专家建议。
5. 结尾：形成课题支撑/深化合作/提升履职质量。

常用句式：

- `调研组一行实地走访了……`
- `座谈会上，……介绍了……情况。`
- `与会人员围绕……展开交流。`
- `调研组表示，将进一步梳理吸收意见建议，形成高质量建言成果。`

写作要点：

- 调研稿要写“为了什么调研”，不是只写“去了哪里”。
- 如果是课题调研，必须落到建言成果。
- 如果是走访盟员单位，要体现关心盟员、凝聚共识、服务发展。

需要用户提供：

- 调研主题。
- 带队领导和参加人员。
- 调研点位。
- 对方介绍内容。
- 座谈观点。
- 后续用途：课题、提案、社情民意、合作项目。

## 六、类型五：人物专访 / 深度人物

适用材料：对盟员专家、学者、艺术家、教师、医生、企业家、参政议政骨干的深度采访。

近期标题样式：

- `铁肩担道义，丹心育桃李——专访上海政法学院副院长郑少华`
- `跨越太平洋的追寻——一位盟员学者与80年前历史的无声对话`
- `上博展览部主任褚馨：现象级大展背后的文化互鉴和专业奉献`
- `蒋薇：让文物“唱”出千年的歌`

文章结构：

1. 场景开头：最近事件、采访现场、忙碌状态、一个动作。
2. 身份定位：本职身份 + 民盟身份 + 核心关键词。
3. 小标题一：人生选择或专业起点。
4. 小标题二：专业成就和代表贡献。
5. 小标题三：履职/社会服务/民盟情怀。
6. 结尾：人物精神和现实意义。

常用写法：

- 先用一个具体场景，不直接罗列简历。
- 用人物原话引出转折。
- 成就用数据、项目、作品、奖项支撑。
- 民盟身份要自然嵌入，不硬贴。

小标题风格：

- `“仗剑走天涯”`
- `“执着地选择教书”`
- `初心：填补历史的空白`
- `征程：与时间赛跑的40天`

写作要点：

- 人物稿不是先进事迹材料，要有故事推进。
- 必须有“人”的细节：动作、语言、选择、困难、转折。
- 不能把采访对象写成完美无缺的宣传符号。
- 民盟公众号的人物稿通常落到“双岗建功”：本职有贡献，盟务/履职有担当。

需要用户提供：

- 人物基本信息：姓名、职务、民盟身份。
- 最近新闻由头：获奖、项目、展览、研究成果、采访契机。
- 关键经历 3-5 个。
- 可引用原话。
- 专业成果和数据。
- 参政议政/社会服务/盟务参与经历。
- 照片说明如有。

## 七、类型六：人物风采 / 获奖喜报

适用材料：盟员获奖、当选、入选、成果发布、作品获奖、荣誉表彰。

近期标题样式：

- `祝贺！盟员柳红荣获第四届“全国创新争先奖状”`
- `祝贺！盟员陈子善获颁“中国现代文学学术贡献荣誉奖”`
- `何祖华院士团队最新研究成果发布，水稻生物育种取得新突破`
- `91岁老戏骨吴云芳又获奖啦！`

文章结构：

1. 开门见山写喜讯。
2. 简述奖项/成果含金量。
3. 介绍人物身份。
4. 写主要贡献。
5. 补充民盟身份和履职情况。
6. 简短收束。

导语模板：

> 近日，……名单公布/……颁奖举行。上海盟员……荣获……。

写作要点：

- 喜报类可短，不必写成完整专访。
- 要解释奖项为什么重要。
- 人物介绍要为“获奖原因”服务。
- 标题可用“祝贺！”“喜报！”但正文不要过度夸张。

需要用户提供：

- 奖项/荣誉准确名称。
- 发布机构。
- 获奖时间。
- 人物身份和民盟身份。
- 获奖原因、代表成果。
- 是否有过往履职或社会贡献。

## 八、类型七：两会 / 建言 / 参政议政类

适用材料：全国两会、上海两会、代表委员建议提案、专题协商、课题成果、社情民意。

近期标题样式：

- `全国两会丨锚定“十五五”蓝图，他们都支了哪些“实招”？`
- `全国两会丨关注“小切口”里的“大民生”，他们用良策温暖民心`
- `全国两会丨一体推进教育科技人才发展，他们这样“破题”……`
- `以高质量研究赋能精准建言，民盟市委第二批课题开题培训举行`

文章结构：

1. 政策背景或会议背景。
2. 代表委员/课题组关注的问题。
3. 问题现状和痛点。
4. 建议内容，尽量具体。
5. 落点：民生、城市治理、高质量发展、制度完善。

多人合集结构：

> 主题总起。
>
> 人物 A：关注问题 + 建议。
>
> 人物 B：关注问题 + 建议。
>
> 人物 C：关注问题 + 建议。
>
> 总结：这些建议体现……

常用句式：

- `某某关注到……`
- `在他/她看来，……`
- `他/她建议，……`
- `围绕……，上海民盟代表委员积极建言。`
- `从“小切口”入手，回应“大民生”。`

写作要点：

- 建言稿要有问题意识，不能只写观点口号。
- 建议要可操作，尽量写清“谁来做、怎么做、做到什么程度”。
- 多人稿要统一颗粒度。
- 标题可以用“实招”“破题”“良策”，但正文要支撑这些词。

需要用户提供：

- 建议/提案标题。
- 提出人及身份。
- 背景问题。
- 数据或案例。
- 具体建议。
- 是否已被采纳、关注、答复。

## 九、类型八：社会服务 / 公益帮扶

适用材料：支教、科普进校园、普法、义诊、捐赠、帮扶、乡村振兴、黄丝带帮教、公益基金项目。

近期标题样式：

- `企业家签约捐赠，闵行区聘专家又招商，“走进”品牌再出新招`
- `为老人留住时光，为孩子点亮课堂——他的公益之路没有终点`
- `科技赋能成长，AI点亮梦想——“MENG芽公益”科普活动走进校园`
- `千亩新绿扎根雪域高原——“民盟沪滇合作生态公益林”落户香格里拉`

文章结构：

1. 背景：为什么做，服务对象是谁。
2. 行动：谁参与，做了什么。
3. 现场：活动场景或受益对象反馈。
4. 成效：人数、项目、数据、长期机制。
5. 价值：专业优势服务社会，民盟担当。

常用表达：

- `以专业力量服务……`
- `把资源送到一线，把服务落到实处。`
- `项目不仅……，也……`
- `为……注入新动能。`

写作要点：

- 社会服务稿必须写“实际效果”。
- 温暖来自事实，不来自形容词。
- 有数据就用数据，没有数据就写具体场景。
- 要体现民盟专业优势，而不是普通公益流水账。

需要用户提供：

- 项目名称和背景。
- 服务对象。
- 参与组织和人员。
- 活动内容。
- 成效数据。
- 受益对象反馈。
- 是否有长期机制。

## 十、类型九：文史 / 纪念 / 盟史类

适用材料：上海民盟 80 周年、盟史人物、传统教育基地、文献档案、口述史、先贤纪念、历史展览。

近期标题样式：

- `跨越太平洋的追寻——一位盟员学者与80年前历史的无声对话`
- `“沪盟八秩，我来打卡”（十）`
- `清明，让我们追思先贤、永续精神`
- `民盟生日快乐！解锁“时光博物馆”，回望沪盟80年`

文章结构：

1. 当下入口：周年、节日、展览、档案发现、人物行动。
2. 历史线索：人物、地点、文献、事件。
3. 叙事展开：按时间或空间推进。
4. 精神解释：民盟传统、合作初心、知识分子担当。
5. 今日回响：对当下组织建设、主题教育、文化传承的意义。

写作要点：

- 文史稿可以更有文学性，但史实必须稳。
- 不要只堆历史资料，要回答“今天为什么讲”。
- 小标题适合用：初心、征程、发现、回响、传承。
- 结尾要落到现实意义，不只怀旧。

需要用户提供：

- 历史主题或人物。
- 可靠史料来源。
- 时间、地点、人物关系。
- 可引用原文或档案。
- 纪念节点。
- 希望落到的现实主题。

## 十一、类型十：预告 / 通知 / 征集

文章结构：

1. 主题一句话。
2. 嘉宾/活动信息。
3. 时间、平台、报名方式。
4. 邀请观看/参与。

写作要点：

- 字数短，通常 100-200 字。
- 标题用问题切入。
- 信息准确比文采重要。
- 健康讲座类要写清医生医院、科室、职称、时间。

需要用户提供：

- 活动主题。
- 嘉宾及职务/职称。
- 时间。
- 平台或地点。
- 报名/预约方式。
- 是否有二维码或说明。

## 十二、标题生成规则

稳妥新闻标题：

- `某某会议召开`
- `某某活动举行`
- `某某赴某地调研`

## 十三、以后给材料时的生成流程

我会按以下流程处理：

1. 先判断文章类型。
2. 列出材料缺口，尤其是人名职务、时间地点、数据、讲话要点。
3. 选择对应结构。
4. 先给 3 个标题备选。
5. 写正文。
6. 检查事实、职务、组织名称、政治表述。
7. 如材料不足，明确标注“待补充”，不编造。
