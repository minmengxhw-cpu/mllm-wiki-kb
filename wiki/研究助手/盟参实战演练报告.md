# 盟参实战演练报告

生成时间：2026-06-17

## 目标

用五个真实工作场景验证 80 周年实战生产系统是否可用：写稿、讲话、文史、核稿、查重。

## 演练一：80 周年活动报道

输入：某庆祝上海民盟80周年主题活动材料。

调用：

```bash
/稿 80周年活动报道
```

使用入口：[[80周年活动报道素材包]]、[[80周年核心材料总表]]。

预期输出：

- 标题 3 个。
- 标准导语。
- 活动流程正文。
- 出席人员、主办承办、数据缺口标 `[待核]`。

## 演练二：主委讲话素材

输入：庆祝大会或专题座谈主题。

调用：

```bash
/稿 80周年主委讲话
```

使用入口：[[80周年主委讲话生产包]]、[[盟参文稿素材：上海民盟80周年主委讲话]]。

预期输出：

- 历史回望、现实成效、工作要求三段结构。
- 可引用文章清单。
- 组织沿革待核项。

## 演练三：文史纪念文章

输入：先贤、旧址、档案或传统教育基地。

调用：

```bash
/史 主题
/稿 80周年文史纪念
```

使用入口：[[80周年文史纪念素材包]]、[[盟史研究深化工作台]]。

预期输出：

- 事实链条。
- 人物/地点/事件关联。
- 文史文章结构。
- 明确区分全国民盟史和上海地方史。

## 演练四：核稿

输入：含“80周年就是民盟成立80周年”“上海民盟市委”“Drive资料自动混入主语料”的测试句。

调用：

```bash
/核 正文
```

验证结果：系统可拦截 80 周年纪念对象、机构名称、Drive 资料边界和来源缺失风险。

## 演练五：选题查重

输入：`沪盟八秩，我来打卡`、`与盟同行会客厅`、`上海民盟组织建立80周年`。

调用：

```bash
/题 选题
```

预期输出：

- 相近历史篇目。
- 可差异化角度：人物、地点、基层组织、履职成果、主题教育。
- 已写过/未充分展开的方向。

## 结论

当前系统已能支撑 80 周年期间的素材准备、初稿生成、史实入口、口径预审和维护闭环。正式发稿前仍需人工补齐活动材料、领导职务、内部口径和权威史实。
