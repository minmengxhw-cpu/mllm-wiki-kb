# 80周年史实核验清单

生成时间：2026-06-17

## 使用场景

所有 80 周年讲话、文史纪念、活动报道、展览说明、选题策划，在发稿前都应按本清单核验。

## 必核事项

| 项目 | 核验问题 | 处理 |
| --- | --- | --- |
| 纪念对象 | 80 周年指上海民盟地方组织建立、全国民盟、抗战胜利还是具体事件 | 未确认则标 `[待核]` |
| 组织名称 | 是否使用“中国民主同盟上海市委员会”或“民盟上海市委” | 避免“上海民盟市委”等错误 |
| 组织沿革 | 成立时间、地点、人物、会议是否准确 | 回权威档案或正式材料 |
| 人物职务 | 是当时职务还是现任职务 | 分开表述 |
| 会议届次 | 区委、基层组织、全委会届次是否准确 | 回原文核对 |
| 活动名称 | 庆祝大会、会客厅、打卡系列、展览名称是否完整 | 按原文或正式方案 |
| 数据成果 | 数字、表彰、项目数量是否有来源 | 无来源标 `[待核]` |
| 历史评价 | 是否把纪念性表述当成史实结论 | 改为线索或待核 |
| 地点 | 旧址、基地、档案馆名称是否准确 | 回地点卡或原文 |

## 高风险主题

- 建盟日期。
- 中国民主政团同盟成立时间。
- 上海民盟地方组织建立时间。
- 民盟被迫解散与恢复活动。
- 李闻事件称谓和人物关系。
- 旧政协、新政协、五一口号表述。
- 上海地方史与全国民盟史混写。

## 核验命令

```bash
kb staff check --file 待核稿件.txt
```

命中 blocker 或 high 风险后，不进入正式发稿。
