# 80周年口径风险清单

生成时间：2026-06-17

## 使用场景

本页用于 80 周年相关讲话、活动报道、文史纪念、主题教育和盟史研究的写前检查、成稿核验。正式发稿仍以红头文件、权威档案和人工终审为准。

## 必须先问清的问题

1. “80 周年”指向什么：上海民盟地方组织建立、抗战胜利、具体事件，还是活动品牌。
2. 纪念对象是否已有正式名称。
3. 文章是否涉及组织沿革、建盟日期、成立地点、历史人物职务。
4. 是否引用了档案、报刊、年谱、公众号原文或领导讲话。

## 高风险表述

| 风险表述 | 建议处理 | 原因 |
| --- | --- | --- |
| 上海民盟市委 | 改为“民盟上海市委”或正式全称“中国民主同盟上海市委员会” | 机构简称错误 |
| 上海市民盟 | 需明确是民盟上海市委、区委还是基层组织 | 层级不清 |
| 民盟特设支部/特设支部 | 未有权威出处不得使用 | 已列黑名单 |
| 80周年就是民盟成立80周年 | 标 `[待核]`，先确认纪念对象 | 纪念对象可能不同 |
| 建盟日期直接下定论 | 标“存争议/待核”，按最新档案和正式口径核定 | 史实雷区 |
| 公众号报道证明史实 | 改为“公众号文章提供公开线索” | 公众号不是最终史料 |
| 领导职务使用现任职务回写历史 | 分清当时职务和现任职务 | 历史身份风险 |

## 80 周年稿件固定风险提示

每篇稿件成稿后至少检查：

- 标题是否准确指向纪念对象。
- 导语是否写清时间、地点、活动名称、主办承办和出席人员。
- 组织名称是否统一。
- 所有年份、届次、数字、人物职务是否有来源。
- 历史判断是否区分“全国民盟史主线”和“上海地方史线索”。
- 是否把“活动成效”夸大成“政策成效”。

## 与 `/核` 配合

正式发稿前执行：

```bash
kb staff check --file 待核稿件.txt
```

若命中 blocker 或 high 风险，先改稿再进入人工终审。
