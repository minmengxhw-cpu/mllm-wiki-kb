# 验收标准

## GitHub 仓库

- main 不含原始资料、OCR 全文、向量、索引、密钥。
- 所有 MiniMax 任务有边界、允许文件、禁止文件、测试命令。
- README 能让新 Agent 明白项目目标和公开边界。

## 本地知识库

- 每个资料层有状态标签：stable、candidate、ocr_unverified、verified。
- 查询结果必须给出来源和可信度。
- OCR 结果不经人工核验不得作为引用定稿。

## 上海盟讯

- 每页至少保留原图路径、OCR 路径、元数据状态、审校状态。
- 引用级页面必须能回看原图。

## 2021 档案

- 人物字段必须区分硬字段、软字段、线索字段。
- 字段必须标注来源页或 OCR 位置。

## 外部网站

- 先登记来源，不默认全文入库。
- 明确版权、来源级别和可引用性。
